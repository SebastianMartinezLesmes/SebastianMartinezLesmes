## 🤖 IAs para trabajar

Herramientas de inteligencia artificial que puedes usar directamente en tu flujo de trabajo como desarrollador, sin costo o con plan gratuito generoso.

### Asistentes de código

| Herramienta | Descripción | Enlace |
|---|---|---|
| **ChatGPT** | El más conocido. Ideal para explicar errores, generar código, refactorizar y documentar. | [chat.openai.com](https://chat.openai.com) |
| **GitHub Copilot** | Autocompletado de código en el editor. Tiene prueba gratuita. | [github.com/features/copilot](https://github.com/features/copilot) |
| **Google Gemini** | Alternativa de Google, integrado con Google Workspace. Muy bueno para análisis de texto y código. | [gemini.google.com](https://gemini.google.com) |
| **Claude (Anthropic)** | Excelente para razonamiento largo, revisión de código y explicaciones detalladas. | [claude.ai](https://claude.ai) |
| **Mistral Le Chat** | IA europea, rápida y gratuita. Buena para tareas de código y texto. | [chat.mistral.ai](https://chat.mistral.ai) |
| **Phind** | IA especializada en programación. Busca en la web y responde con contexto técnico. | [phind.com](https://www.phind.com) |
| **Codeium** | Autocompletado de código gratuito para VS Code, JetBrains y más. | [codeium.com](https://codeium.com) |

### Generación de imágenes y diseño

| Herramienta | Descripción | Enlace |
|---|---|---|
| **Adobe Firefly** | Generación de imágenes con IA, integrado con herramientas Adobe. Plan gratuito disponible. | [firefly.adobe.com](https://firefly.adobe.com) |
| **Canva AI** | Diseño con asistencia de IA. Ideal para crear assets visuales rápido. | [canva.com](https://www.canva.com) |
| **Remove.bg** | Elimina el fondo de imágenes automáticamente con IA. | [remove.bg](https://www.remove.bg) |

### Productividad y automatización

| Herramienta | Descripción | Enlace |
|---|---|---|
| **Notion AI** | Toma notas, genera documentación y organiza proyectos con ayuda de IA. | [notion.so](https://www.notion.so) |
| **Perplexity AI** | Motor de búsqueda con IA. Responde preguntas técnicas con fuentes verificadas. | [perplexity.ai](https://www.perplexity.ai) |
| **v0 by Vercel** | Genera componentes UI en React/Tailwind a partir de una descripción en texto. | [v0.dev](https://v0.dev) |
| **Bolt.new** | Genera aplicaciones web completas con IA directamente en el navegador. | [bolt.new](https://bolt.new) |

## 🧪 Playgrounds — testea código online

Ejecuta y prueba código directamente en el navegador, sin instalar nada. Útil para prototipar, aprender o compartir snippets.

### Multi-lenguaje

| Plataforma | Lenguajes soportados | Destacado | Enlace |
|---|---|---|---|
| **Replit** | +50 lenguajes (Python, JS, C++, Java, Go, Rust...) | Entorno completo con terminal, archivos y colaboración en tiempo real | [replit.com](https://replit.com) |
| **JDoodle** | +70 lenguajes | Simple y rápido, sin registro necesario | [jdoodle.com](https://www.jdoodle.com) |
| **OneCompiler** | +60 lenguajes | Interfaz limpia, soporte para stdin, sin registro | [onecompiler.com](https://onecompiler.com) |
| **Programiz** | Python, C, C++, Java, JS, SQL... | Ideal para aprender, con ejemplos integrados | [programiz.com/compiler](https://www.programiz.com/online-compiler) |
| **Ideone** | +60 lenguajes | Permite compartir código con link público | [ideone.com](https://ideone.com) |
| **Glot.io** | +40 lenguajes | Open source, minimalista | [glot.io](https://glot.io) |

### Frontend (HTML / CSS / JS)

| Plataforma | Descripción | Enlace |
|---|---|---|
| **CodePen** | El más popular para frontend. Ideal para compartir demos visuales. | [codepen.io](https://codepen.io) |
| **StackBlitz** | Entorno completo de Node.js en el navegador. Soporta React, Angular, Vue, etc. | [stackblitz.com](https://stackblitz.com) |
| **CodeSandbox** | Sandbox completo para proyectos frontend modernos con npm. | [codesandbox.io](https://codesandbox.io) |
| **JSFiddle** | Clásico para probar HTML, CSS y JS rápidamente. | [jsfiddle.net](https://jsfiddle.net) |
| **PlayCode** | Editor en tiempo real muy rápido para JS/TS. | [playcode.io](https://playcode.io) |

### Por lenguaje específico

| Lenguaje | Plataforma | Enlace |
|---|---|---|
| **Python** | Python Tutor — visualiza la ejecución paso a paso | [pythontutor.com](https://pythontutor.com) |
| **SQL** | SQLiteOnline — ejecuta queries SQL en el navegador | [sqliteonline.com](https://sqliteonline.com) |
| **SQL** | DB Fiddle — prueba queries en MySQL, PostgreSQL, SQLite | [dbfiddle.uk](https://dbfiddle.uk) |
| **Regex** | Regex101 — construye y depura expresiones regulares | [regex101.com](https://regex101.com) |
| **TypeScript** | TypeScript Playground oficial | [typescriptlang.org/play](https://www.typescriptlang.org/play) |
| **Rust** | Rust Playground oficial | [play.rust-lang.org](https://play.rust-lang.org) |
| **Go** | Go Playground oficial | [go.dev/play](https://go.dev/play) |
| **Java** | JShell Online | [tryjshell.org](https://tryjshell.org) |

## 📧 Privacidad — correos y números temporales

### Correos temporales

Crea direcciones de correo desechables para registrarte en sitios sin exponer tu correo real. Los mensajes llegan en segundos y caducan automáticamente.

| Servicio | Enlace |
|---|---|
| **Temp-Mail** | [temp-mail.org](https://temp-mail.org/es/) |
| **Guerrilla Mail** | [guerrillamail.com](https://www.guerrillamail.com) |
| **10 Minute Mail** | [10minutemail.com](https://10minutemail.com) |

### Números de celular temporales

Recibe SMS de verificación online sin usar tu número real. Útil para registros rápidos.

| Servicio | Enlace |
|---|---|
| **Temp Number** | [temp-number.org](https://temp-number.org/app/) |
| **Receive SMS Online** | [receive-smss.com](https://receive-smss.com) |

| 📐 **WhatFont** | Identifica la fuente tipográfica de cualquier texto en una página web. | [Chrome Web Store](https://chrome.google.com/webstore/detail/whatfont/jabopobgcpjmedljpbcaablpmlmfcogm) |
